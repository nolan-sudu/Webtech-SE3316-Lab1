# Objective

- Create a HTML document that shows data for the 10 butterfly types from the data provided.

# Features

- Displays a search box at the top of the page to search by any word in the name or description.

- Uses JavaScript to validate input for the search box. It accepts no more than 20 characters and only accept letters A-Z (upper or lower case).

- “Enter” key or the “Search” button displays a pop-up message with common names of all matches (up to 5).

- Displays a drop-down list containing the scientific names. When a name is selected, a pop-up message with the common name and description of the butterfly type shows.

- A footer which gives credit to the data with a clickable link to the source.
